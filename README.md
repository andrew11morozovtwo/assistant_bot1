# Telegram-бот для телеграмм-канала https://web.telegram.org/k/#@its_not_canal "Это не канал" для пересказа и анализа контента с использованием AI

Этот проект представляет собой Telegram-бота, который помогает быстро получать пересказы и анализировать различное содержимое, отправленное пользователем. Бот использует API OpenAI для перефразирования текста, создания описаний изображений и транскрибации аудио, а также поддерживает извлечение текста из веб-страниц.

## Описание

Этот бот предназначен для быстрого получения информации из различных источников. Он может быть полезен для пересказа статей, анализа изображений и понимания содержания аудио-сообщений. Просто отправьте боту текст, ссылку, изображение или аудио, и он предоставит соответствующий результат.

## Возможности

*   **Перефразирование текста:** Бот использует OpenAI для перефразирования текстовых сообщений.
*   **Извлечение текста из веб-страниц:** Бот может извлекать текст из веб-страниц по URL.
*   **Описание изображений:** Бот использует OpenAI Vision API для создания описаний изображений.
*   **Транскрибация аудио:** Бот использует OpenAI Whisper API для транскрибации аудио- и голосовых сообщений.
*   **Обработка URL в подписях:** Бот способен находить и обрабатывать URL-адреса, содержащиеся в подписях к медиафайлам, извлекая контент для более полного анализа.
*   **Поддержка различных типов сообщений:** Бот обрабатывает текст, фото, видео, голосовые и аудио сообщения, а также опросы.
*   **Логирование:** Бот ведет логи всех запросов и ответов в CSV-файл для анализа и отладки.
*   **История разговоров:** Бот сохраняет историю разговоров с каждым пользователем для поддержания контекста.

## ⚡ Быстрый старт

### 1. Получение токенов

#### Telegram Bot Token
1. Отправьте `/newbot` [@BotFather](https://t.me/BotFather)
2. Следуйте инструкциям для создания бота
3. Сохраните полученный токен

#### OpenAI API Key
1. Зарегистрируйтесь на [OpenAI Platform](https://platform.openai.com/)
2. Перейдите в раздел API Keys
3. Создайте новый API ключ

### 2. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
```

### 3. Запуск бота

```bash
python bot.py
```

## 📚 API Документация

### Основные функции

#### `process_message(message, user_message, message_type, chat_id)`
Основная функция обработки сообщений с использованием OpenAI API.

**Параметры:**
- `message` - объект сообщения Telegram
- `user_message` - текст сообщения пользователя
- `message_type` - тип сообщения
- `chat_id` - ID чата

#### `handle_photo_message(message)`
Обработчик изображений с использованием OpenAI Vision API.

#### `handle_pdf_message(message)`
Обработчик PDF документов с извлечением и анализом текста.

#### `handle_voice_message(message)`
Обработчик голосовых сообщений с транскрибацией через Whisper API.

### Системный промпт

Бот использует специализированный системный промпт для работы в качестве администратора канала:

```python
system_prompt = """
Вы бот-администратор в телеграм-канале 'Это не канал'. Ваша задача — пересказывать на русском языке 
подписчикам материалы, присылаемые в канал. Формируйте краткий (не более 3000 знаков) и интересный пересказ, 
ориентируясь на следующие принципы:

1. Прочитайте пост. Если в посте указана ссылка, предположите, что она содержит дополнительную информацию.
2. Пересказ оформляйте структурно:
   - Введение: кратко объясните, о чем материал и почему он важен.
   - Основная часть: изложите ключевые моменты материала простым языком.
   - Заключение: сделайте выводы или задайте вопрос для вовлечения.
3. Используйте эмодзи для акцентирования ключевых моментов.
4. Указывайте источник информации в конце текста.
"""
```
