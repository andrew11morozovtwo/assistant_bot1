# Telegram-бот для телеграмм-канала https://web.telegram.org/k/#@its_not_canal "Это не канал" для пересказа и анализа контента с использованием AI

Этот проект представляет собой Telegram-бота, который помогает быстро получать пересказы и анализировать различное содержимое, отправленное пользователем. Бот использует API OpenAI для перефразирования текста, создания описаний изображений и транскрибации аудио, а также поддерживает извлечение текста из веб-страниц.

## Описание

Этот бот предназначен для быстрого получения информации из различных источников. Он может быть полезен для пересказа статей, анализа изображений и понимания содержания аудио-сообщений. Просто отправьте боту текст, ссылку, изображение или аудио, и он предоставит соответствующий результат.

## Возможности

*   **Перефразирование текста:** Бот использует OpenAI для перефразирования текстовых сообщений.
*   **Извлечение текста из веб-страниц:** Бот может извлекать текст из веб-страниц по URL.
*   **Описание изображений:** Бот использует OpenAI Vision API для создания описаний изображений.
*   **Транскрибация аудио:** Бот использует OpenAI Whisper API для транскрибации аудио- и голосовых сообщений.
*   **Обработка URL в подписях:** Бот способен находить и обрабатывать URL-адреса, содержащиеся в подписях к медиафайлам, извлекая контент для более полного анализа.
*   **Поддержка различных типов сообщений:** Бот обрабатывает текст, фото, видео, голосовые и аудио сообщения, а также опросы.
*   **Логирование:** Бот ведет логи всех запросов и ответов в CSV-файл для анализа и отладки.
*   **История разговоров:** Бот сохраняет историю разговоров с каждым пользователем для поддержания контекста.

## Как запустить

1.  **Клонируйте репозиторий:**
    ```
    git clone https://github.com/ваш_пользователь/ваш_репозиторий.git
    cd ваш_репозиторий
    ```
2.  **Установите необходимые зависимости:**
    ```
    pip install -r requirements.txt
    ```
    *Убедитесь, что у вас установлен Python 3.7 или выше.*
3.  **Настройте переменные окружения:**
    *   В Google Colab используйте раздел "Secrets" для безопасного хранения токена Telegram-бота и API-ключа OpenAI.
    *   Добавьте следующие переменные:
        *   `TELEGRAM_BOT_TOKEN`: Токен вашего Telegram-бота.
        *   `OPENAI_API_KEY`: Ваш API-ключ OpenAI.
        *   Получить значения переменных можно с помощью библиотеки `google.colab`:
        ```
        from google.colab import userdata
        TELEGRAM_BOT_TOKEN = userdata.get('TELEGRAM_BOT_TOKEN')
        OPENAI_API_KEY = userdata.get('OPENAI_API_KEY')
        ```
    *  *Внимание! Не храните API-ключи и токены в коде напрямую!*

4.  **Запустите бота:**
    ```
    python ваш_бот.py
    ```
    *Замените `ваш_бот.py` на имя вашего основного файла с кодом бота.*

## Требования

*   Python 3.7+
*   Библиотеки из `requirements.txt`
*   Аккаунт OpenAI с доступом к API (включая Whisper API и, опционально, Vision API)
*   Токен Telegram-бота
*   Google Colab (для использования Colab Secrets)

## Структура кода
*   **Основные библиотеки**: `telebot`, `openai`, `google.colab`, `logging`, `csv`, `bs4`, `requests`.
*   **Функция `process_url_in_text(text, bot, chat_id)`**: Ищет URL в тексте и извлекает содержимое веб-страницы.
*   **Обработчики сообщений**:
    *   `handle_text_message(message)`: Обрабатывает текстовые сообщения.
    *   `handle_photo_message(message)`: Обрабатывает фотографии и получает описания изображений с помощью OpenAI Vision API.
    *   `handle_video_message(message)`: Обрабатывает видео.
    *   `handle_voice_message(message)`: Обрабатывает голосовые сообщения и транскрибирует их с помощью OpenAI Whisper API.
    *   `handle_audio_message(message)`: Обрабатывает аудио сообщения и транскрибирует их с помощью OpenAI Whisper API.
    *   `handle_poll_message(message)`: Обрабатывает опросы.
*   **Функция `process_message(message, user_message, message_type, chat_id)`**: Основная функция для обработки сообщений пользователей, взаимодействует с OpenAI для генерации ответов.
