# Телеграм-бот для быстрого просмотра научных статей без VPN

Этот проект представляет собой телеграм-бота, который помогает быстро просматривать научные статьи, размещенные на веб-сайтах, которые могут быть недоступны в некоторых регионах. Бот использует API OpenAI для создания кратких пересказов материалов, отправленных пользователями, с добавлением эмодзи для акцентов на ключевых моментах.

## Описание

Я создал этого бота для себя, чтобы иметь возможность читать статьи с недружественных сайтов без необходимости использовать VPN. Бот помогает извлекать текст с веб-страниц и автоматически генерировать краткие пересказы на русском языке. Всё, что нужно — это отправить ссылку на статью в чат с ботом, и он ответит пересказом.

## Возможности

- Поддержка извлечения текста с веб-страниц (извлекает основную информацию с URL).
- Автоматическое создание пересказов научных материалов с использованием OpenAI.
- Легкость в использовании: отправляйте текст или ссылку, и получите пересказ или описание.
- Поддержка различных типов сообщений: текст, фото, видео, голосовые и аудио сообщения.

## Как запустить

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш_пользователь/ваш_репозиторий.git
   ```
   
2. Установите все зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. В файле `bot.py` замените следующие значения:
   - `API_TOKEN = '_______'` — на ваш токен Telegram-бота, который можно получить через [BotFather](https://core.telegram.org/bots#botfather).
   - `api_key="_____"` — на ваш API-ключ OpenAI, если вы планируете использовать GPT-3.5 для генерации пересказов.

4. Запустите бота:
   ```bash
   python bot.py
   ```

## Пример использования

1. Отправьте боту ссылку на научную статью. Например:
   ```
   https://example.com/scientific_article
   ```

2. Бот извлечет текст с указанной страницы и сгенерирует его краткий пересказ.

## Логирование

Все сообщения, включая пользовательские и ответы от AI, записываются в файл логов `telegram_bot_logs.csv`. Логи включают следующие данные:
- `chat_id` — ID чата.
- `datetime` — время сообщения.
- `message` — сообщение пользователя.
- `message_type` — тип сообщения (текст, фото, видео и т. д.).
- `ai_response` — ответ от AI.
